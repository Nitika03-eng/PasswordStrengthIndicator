# 🔐 Password Strength Indicator

A simple and interactive Password Strength Indicator built with **HTML**, **CSS**, and **JavaScript**. This project visually evaluates and displays the strength of a user's password based on length, use of uppercase/lowercase characters, numbers, and special characters.

---

## 🌟 Features

- Real-time password strength feedback
- Visual strength bar (Weak, Medium, Strong)
- Validates:
  - Password length
  - Use of uppercase/lowercase letters
  - Use of numbers and symbols
- Clean and responsive UI

## 🧠 How It Works

1. User types a password into the input field.
2. A script evaluates the password in real-time based on:
   - Length of the password
   - Use of uppercase/lowercase letters
   - Use of numbers
   - Use of special characters
3. A colored strength bar and/or text updates accordingly.
